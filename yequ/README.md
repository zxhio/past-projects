## YEQU ##
- 启动
    - BIOS 将第一扇区 `boot` 移动至 `0x7C00`，
    - `boot` 利用 `int 0x13` 读取软盘内容 `setup` 至 `0x9000`, 并跳转至 `setup`
    - `setup` 将 `kernel` 读取至内存中，再移动至 `0x00` 并跳转 至 `head`
- VGA 显示
    - 显存映射到内存地址 `0xB8000` 开始的空间上，每两个字节代表一个字符，窗口大小为 `80 x 25`，vga 实现 `printk` 函数（// TODO： 串口，并口）
